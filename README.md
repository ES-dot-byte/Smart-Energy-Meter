# Smart Energy Meter with Theft Detection

## Project Overview
This project presents a smart energy meter system using Arduino UNO and ESP32.

## Features
- Real-time energy monitoring
- Prepaid recharge using WiFi
- Theft detection system
- Automatic power cutoff
- LCD display for units and cost

## Technologies Used
- Arduino UNO
- ESP32
- IoT (WiFi)
- Embedded Systems
